# Pinnacle Studio

A modern architectural studio website showcasing brutalist elegance and sustainable innovation. Built with Node.js, Express, and vanilla JavaScript.

## Features

- **Multi-page Architecture**: Home, Projects, Studio, Services, and Contact pages
- **Modern Design**: Pure black background with white typography and premium animations
- **Responsive Layout**: Mobile-first design with smooth interactions
- **Contact Form**: Functional contact form with validation
- **Progress Bar**: Scroll progress indicator with mix-blend-mode effects
- **Mobile Menu**: Hamburger navigation for mobile devices

## Tech Stack

- **Backend**: Node.js with Express.js
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Styling**: Custom CSS with CSS variables and animations
- **Icons**: Lucide React icons
- **Fonts**: Manrope & Syncopate from Google Fonts

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Start with auto-restart
npm run dev
```

## Design Philosophy

Clean, minimalist aesthetic with a focus on typography and spacing. The design emphasizes the architectural studio's brutalist elegance through a stark black background with precise white typography and subtle animations.

## Pages

- **Home**: Hero section with brand introduction
- **Projects**: Portfolio showcase with project cards
- **Studio**: About the architectural studio and philosophy
- **Services**: Services offered by the studio
- **Contact**: Contact form and information
